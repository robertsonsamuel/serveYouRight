How to use this app.

  1. Create a new menu by giving it a name and clicking "Create"
  2. Click the edit pencil to add items to the menu or edit existing ones.
  3. Give your employee the store code and have them register with the storce code listed.
  4. Have your employee download the app for iOS or Android, login and begin to take orders.
  5. Visit the kitchen and enter your store code to see orders come in real time.
  6. When an order is complete simply click or tap "Complete Order" to notify the server!
